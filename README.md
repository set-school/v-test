# VerbSprint

Два интерактивных теста по 100 частотным английским глаголам:

- выбор правильной формы Past Simple;
- выбор английского инфинитива по русскому значению.

В обоих тестах есть таймер, звуковая обратная связь, одноразовый выбор ответа и итоговый разбор ошибок.

Страница не требует сборки или зависимостей. Для публикации включите GitHub Pages в настройках репозитория: **Settings → Pages → Deploy from a branch**, затем выберите ветку и папку `/ (root)`.
